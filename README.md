## 🌤️ Day 1 — SkyScan Weather
A modern and responsive weather application built using **HTML**, **CSS**, and **JavaScript**. SkyScan Weather allows users to search for any city and view real-time weather information powered by the **OpenWeather API**.

> 🚀 Day 1 of my **30 Days, Build Something New** challenge.

---

## 📸 Preview

URL: https://sushmita25dubey.github.io/SkyScan-Weather-App/

---

## ✨ Features

- 🔍 Search weather by city name
- 🌡️ Real-time temperature
- ☁️ Weather condition
- 💧 Humidity
- 🌬️ Wind Speed
- 📊 Atmospheric Pressure
- 🤗 Feels Like Temperature
- 📅 Current Date
- 🎨 Modern Dark UI
- 📱 Fully Responsive Design
- ⚡ Fast and Lightweight

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- OpenWeather API
- Font Awesome
